<h1>📋 Task Manager Web Application</h1><br>

<h2>📌 Description</h2>
This project is a simple Task Manager Web Application developed using HTML, CSS, and JavaScript. 
It allows users to efficiently manage their daily tasks with a clean and responsive interface.

<h2>🚀 Features</h2>
<ul>
<li><b>Add new tasks</b></li>
<li><b>Display task list</b></li>
<li><b>Mark tasks as completed</b></li>
<li><b>Delete tasks</b></li>
<li><b>Dynamic updates without page reload</b></li>
</ul>
<h2>🛠️ Technologies Used</h2>
<ul>
<li><b>HTML</b></li>
<li><b>CSS</b></li>
<li><b>JavaScript</b></li>
</ul>
<h2>📁 File Structure</h2>
<ul>
<li>index.html   → Structure of the application  </li>
<li>style.css    → Styling and layout  </li>
<li>script.js    → Functionality and DOM manipulation  </li>
</ul>
<h2>⚙️ Functionality</h2>
<ul>
<li>Users can add tasks using an input field</li>
<li>Tasks are displayed dynamically on the screen</li>
<li>Tasks can be marked as completed</li>
<li>Tasks can be deleted instantly</li>
<li>All actions update the UI without refreshing the page</li>
</ul>
<h2>🎯 Objective</h2>
To demonstrate the use of DOM manipulation, event handling, and responsive design using core web technologies.
